iDialog AngularJS Example

How to use it? Add idialog module to your application. For example:
angular.module('my-cool-app', ['idialog']);

Want to use animations?
angular.module('my-cool-app', ['idialog', 'ngAnimate']);

Don't forget, that ngAnimate need to be connected by including a script to your page.

iDialog is a very simple AngularJS module to show any content in a popup window. Content can be an inline template or can be loaded from a server.

  • Click this link to see a very simple dialog window.
  • Windows can contains any content or angular directives. Each dialog window has an own $scope. See an example.
  • You can add an any class to iDialog. Example.
  • Click this link to load content from a server.

You can use idialog inside angular controllers and directives. Include $idialog service to your controller or directive and use it:
$idialog('idialog-3');

You can pass options to dialog window. Now it's class only, that add any class name to window tag.
